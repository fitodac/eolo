# Color

**Eolo** viene en 3 sabores:

- Primary
- Secondary
- Alternative

Puedes usar clases de color tal como lo haces en Tailwind:

```css
/* Background color */
.bg-primary

/* Font color */
.text-primary

/* Box shadow */
.shadow-primary
```
