# Contenido

Por defecto, si utilizas elementos como listas no ordenadas `<ul>` o encabezados `<h1>`, no generarán ningún estilo. Para lograrlo, pueden ser envueltos dentro de un div con la clase `.content`.

<!-- By default, if you use elements such as unordered lists `<ul>` or headers `<h1>`, they won't generate any styling. To achieve that, they can be wrapped inside a div with the class `.content`. -->

	## Textos inline
	<InlineText />
	## Párrafos
	<Paragraph />
	## Lista desordenada
	<UnorderedList />
	## Listas ordenadas
	<OrderedList />
</div>
